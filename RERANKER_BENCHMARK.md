## Key Observations

1. **CPU → GPU FP32**: 1.0x faster (14.9 → 14.7 ms/query)
2. **FP32 → FP16**: 40% latency reduction (14.7 → 8.8 ms)
3. **Scaling**: Latency from 7.0 ms (10 candidates) to 25.4 ms (60 candidates)

## Methodology

- **Reranker**: Cross-encoder scores each (query, document) pair independently
- **End-to-end**: ChromaDB bi-encoder retrieval + cross-encoder reranking
- **Warmup**: 3 queries discarded before timing
- **Repetitions**: 3 full passes, mean ± std reported
- **Timing**: `torch.cuda.synchronize()` for accurate GPU measurement
- **Isolation**: GPU memory cleared between configurations
